# WanderLust

WanderLust is a web application for creating, managing, and exploring travel listings. Users can view destinations, add new listings, update existing information, and remove listings through a simple and intuitive interface.

## Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* EJS
* Bootstrap

## Features

* Create, read, update, and delete listings
* Server-side validation
* RESTful routing
* MongoDB integration
* Custom error handling
* Responsive user interface

## Installation

```bash
git clone https://github.com/pratyushPanda23/WanderLust.git
cd WanderLust
npm install
npm start
```
