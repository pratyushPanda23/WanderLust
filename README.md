# WanderLust 🌍

A full-stack travel listing web application inspired by Airbnb, where users can explore, create, edit, and manage travel destination listings.

##  Features

* View all travel listings
* Add new listings
* Edit existing listings
* Delete listings
* Server-side form validation
* Error handling using custom middleware
* Responsive and user-friendly interface
* RESTful routing architecture
* MongoDB database integration

##  Tech Stack

### Frontend

* HTML
* CSS
* JavaScript
* EJS (Embedded JavaScript Templates)

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

### Other Tools & Packages

* Method Override
* EJS-Mate
* Express Middleware
* Custom Error Handling

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/pratyushPanda23/WanderLust.git
```

### Navigate to Project Directory

```bash
cd WanderLust
```

### Install Dependencies

```bash
npm install
```

### Start the Server

```bash
node app.js
```

or

```bash
nodemon app.js
```

### Open in Browser

```text
http://localhost:8080/listings
```
